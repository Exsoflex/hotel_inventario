Proceso de Insalacion

1. Instalar XAMPP.

2. Copiar la carpeta hotel_inventario a:

C:\xampp\htdocs\

3. Iniciar Apache y MySQL.

4. Abrir phpMyAdmin.

5. Crear la base de datos hotel_inventario.

6. Importar hotel_inventario.sql de la carpeta "Pagina_Inventario"

7. Abrir:   http://localhost/hotel_inventario
